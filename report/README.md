# Dashboard de Control de Gastos

Este dashboard fue desarrollado en Power BI sobre los datasets procesados previamente en Python.

## Objetivo del dashboard

- Visualizar el gasto mensual y su evolución
- Identificar cuentas que explican el desvío
- Destacar variaciones atípicas (outliers)
- Priorizar cuentas a justificar según política definida

## Nota sobre los datos

El archivo Power BI (.pbix) no se incluye por contener información sensible del modelo y las fuentes de datos.

Las capturas reflejan la estructura, lógica y narrativa del dashboard.
